Movie Searching Website | React.js Project

This project combines the power of React, Tailwind CSS, and Redux Toolkit to create an engaging web application for movie enthusiasts.
Here are some of the key features I implemented:

Responsive Design

I utilized Tailwind CSS to ensure that the web app looks great and functions seamlessly on various devices.

Authentication

I've successfully implemented Firebase for user authentication, enabling users to sign up, log in, and securely store their data in a Firebase database.

Data Integration

The app seamlessly integrates with the TMDB API to fetch and display movie data. TMDB API

State Management

I harnessed the capabilities of Redux Toolkit (RTK) to efficiently manage and maintain the data layer within the application. Redux Toolkit

Routing

For seamless navigation, I incorporated react-router-dom for routing purposes.

GPT Search & Shimmer UI

Leveraging the power of OpenAI APIs, I implemented a natural language search feature using GPT, allowing users to search for movies in a more conversational manner.

I've also implemented a shimmer UI effect for a visually appealing page-loading experience.

Clean Code Practices

To ensure maintainability, I utilized React hooks and created custom hooks, contributing to cleaner and more organized code.

Multi-Language Support

The app features multi-language support, making it accessible to a wider audience.

Component Reusability

I designed and built reusable components to enhance code efficiency and promote scalability.
